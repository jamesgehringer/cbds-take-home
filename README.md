# How to run

Both exersice1.py and exercise2.py will run without any input from the root directory:

## Example: 
- python .\src\exercise1.py
- python .\src\exercise2.py

Both will save their images to the results folder. There are static pngs for ease. There are additionally interactive html plots that will open in any browser.

Code is commented as requested. More specific thoughts with findings and embedded images or in the root directory. For the question about geopolitical events, see historical_event_findings.md. For the question exploring gdp, see gdp_findings.md. Additionally, my notes as I worked through the problems were documented in notes.md for additional reference. There is also an additional folder of screenshots that I took throughout the development that are embedded in the markdown files.
